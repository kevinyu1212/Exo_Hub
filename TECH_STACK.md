# 🚀 ExoHub Technology Specification (2026.04.22)

## Frontend Ecosystem
- **Core**: Next.js 14, TypeScript, NPM Workspaces
- **State**: Zustand, Tanstack Query, XState
- **UI**: Tailwind CSS, Framer Motion, React Aria
- **Charts**: Recharts

## Infrastructure & Resilience
- **Real-time**: Socket.io (WebSocket)
- **Persistence**: IndexedDB (idb), Next-PWA
- **Security**: Zod, Crypto-JS, React Error Boundary
- **Optimization**: Sharp, React-Lazyload, NProgress

## Developer Experience (DX)
- **Testing/Dev**: React Query Devtools, Axe-core, Custom Dev-Suite
- **Documentation**: Markdown-based Architecture Guide
